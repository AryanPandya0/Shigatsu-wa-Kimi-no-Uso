# 🌸 Shigatsu wa Kimi no Uso (Your Lie in April) Tribute

> "A spring without you is coming."

A cinematic, highly aesthetic, and emotionally immersive tribute website dedicated to the masterpiece anime *Your Lie in April*. This project captures the essence of the story through music, motion, and visual storytelling.

---

## ✨ Features

- **🌸 Interactive Petal System**: A custom HTML5 Canvas animation that brings the iconic cherry blossoms to life, reacting to your mouse movements.
- **🎵 Atmospheric Audio**: Integrated with Howler.js to provide a seamless musical experience, featuring Chopin's *Ballade No. 1 in G Minor*.
- **🎭 Cinematic Animations**: Powered by GSAP and ScrollTrigger for smooth, scroll-based transitions and a "living memory" feel.
- **💎 Glassmorphism Design**: A modern, premium UI/UX inspired by the anime's pastel color palette.
- **✉️ Final Letter Experience**: An interactive typewriter effect that recreates Kaori's heartbreaking final letter to Kousei Arima.

## 🛠️ Built With

- **HTML5 & CSS3** (Vanilla for maximum performance)
- **JavaScript** (ES6+)
- **[GSAP](https://greensock.com/gsap/)** - For professional-grade motion design
- **[Howler.js](https://howlerjs.com/)** - For robust audio handling
- **Google Fonts** (Playfair Display, Inter, Dancing Script)
